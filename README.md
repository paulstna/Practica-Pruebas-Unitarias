# Practica-Pruebas-Unitarias

## Autor
Nombre: Paul Salas
Materia: Verificación y Validación de Software

## Descripción
Este repositorio contiene una práctica de pruebas unitarias para la materia de Verificación y Validación de Software. El objetivo es demostrar el uso de pruebas unitarias en un proyecto Java utilizando Maven.

## Pasos para clonar el proyecto y ejecutar las pruebas unitarias
1. Clona el repositorio desde GitHub:
   ```
   git clone https://github.com/paulstna/Practica-Pruebas-Unitarias
   ```

2. Navega al directorio del proyecto:
   ```
   cd Practica-Pruebas-Unitarias
   ```

3. Navega al subdirectorio del proyecto:
   ```
   cd practica-pruebas-unitarias
   ```

4. Ejecuta las pruebas unitarias con Maven:
   ```
   mvn test
   ```
